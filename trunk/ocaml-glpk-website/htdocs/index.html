ocaml-glpk

ocaml-glpk are OCaml bindings to GLPK (GNU Linear Programming Kit) which is a package is intended for solving large-scale linear programming (LP), mixed integer programming (MIP), and other related problems.

You can either browse the module's documentation or download the library.

If you've got any question, bug, problem or patch, feel free to mail me.
